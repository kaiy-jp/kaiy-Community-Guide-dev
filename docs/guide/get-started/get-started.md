# kaiy.coの概要
kaiy.coとはどのようなサービスなのでしょうか？

kaiy.coとは、Web 3.0時代の新しいマイクロコミュニティサービスです。このサービスでは、友達や同じ会社の人たち、家族と、テレカンファレンスをしながら楽しくおしゃべりをしたり、飲み会をしたり、ライブを見たり、一緒にワイワイとゲームをしたりすることができます。

また、kaiy.coの中に存在する好きなお店での買い物を家にいながらにしてできます。
